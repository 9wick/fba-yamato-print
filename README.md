# fba-yamato-print

Amazon から出力されるクロネコヤマト用のPDFを、クロネコヤマトのA5横印刷用紙に合わせて印刷するための簡易ツールです。[Webページ](https://9wick.github.io/fba-yamato-print/) をローカルで開くだけで使えます。

## 使い方

1. [Webページ](https://9wick.github.io/fba-yamato-print/)  をブラウザで開く
2. PDFをドラッグ＆ドロップ（または右上のファイル選択）
3. Scale/Offsetを調整してプレビュー位置を合わせる
4. 印刷ボタンでA5横・余白0で印刷

## メモ

- PDFの描画にはCDNのpdf.jsを使うため、初回はネットワーク接続が必要です。
